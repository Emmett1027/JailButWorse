Join Our Server

Joining JailButWorse is simple! Just follow these steps:

  1. Open Minecraft Java Edition (1.21.4+).
  2. Go to “Multiplayer” → “Add Server”.
  3. Set the server IP to JailButWorse.minehut.gg.
  4. Click “Done” and then “Join Server”.
